# URL-To-QRcode
> 練習接Google所提供QRcode api來製作，輸入網址後送出即可生成。

![](https://i.imgur.com/BD3QRA7.png)
## 使用技術&練習技術
- JavaScript(ES6)
- scss
- HTML
- ESlint(standard)
